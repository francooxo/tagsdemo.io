# 🎨 Advanced HTML Tags Exploration - Day 1

This is a simple web page built for my Day 1 challenge. It explores and demonstrates how to use advanced HTML semantic and interactive tags.

---

## ⭐ Features
* **Semantic Elements:** Uses tags like `<abbr>`, `<blockquote `>`, and `<cite>` for proper text styling and meaning.
* **Interactive Elements:** Features `<details>` and `<summary>` tags to make content collapsible.
* **Inline Snippets:** Uses the `<code>` and `<mark>` tags to highlight code and key text.

## 🛠️ Built With
* HTML5
