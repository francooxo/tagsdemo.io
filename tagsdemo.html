1. The <dfn> Tag

Use definition to mark the first occurrence of a term you're defining. It helps indicate where a term is formally introduced.

Example: Semantic HTML is HTML that conveys meaning about the content, not just presentation.

2. The <kbd> Tag

The <kbd> element represents keyboard input or commands a user should type.

Example: To save your document, press Ctrl + S.

3. The <samp> Tag

The <samp> element represents sample output from a program or system.

Example: When you run the command, you'll see: Hello, World!

4. The <progress> Tag

The <progress> element displays a progress bar showing completion status.
